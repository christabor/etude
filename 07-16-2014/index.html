A simple yet powerful modal widget.

Here are some reasons why.

  • Allows for N nested modals - no limit!
  • Easy to style, allows for header, footer, body (similar to bootstrap)
  • Animations can easily tacked on, (example used: animate.css)
  • Customizeable via api
  • Simple design allows for toggling between modals when clicking on any modal target.
  • Keeps track of z-index, so you don't have too.
  • Multiple sizing options
  • Mobile friendly!
