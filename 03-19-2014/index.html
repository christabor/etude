Custom Sigil maker!

A sigil is a type of magical instrument used to (supposedly) create spells that harness the users intentions, by encoding them into a visual abstraction that uh... looks cool!

Export

Steps: how it actually “works”

  1. Write down what you want to wish for/have.
  2. Take out all vowels
  3. Remove all duplicate consonants
  4. Make some weird art by re-arranging and resizing the letters! Fame and fortune, here we come!
