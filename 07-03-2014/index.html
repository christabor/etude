A proof-of-concept graph content editor

Another (long-term) project I am working on explores the idea of creating tools that allow one to visualize an idea or problem as a set of connected nodes (aka "graph"). Editing and persisting data is also very important, so this is an example of how it might be done.

Click on a node to view it and edit the content. Click again to see that it was updated.
